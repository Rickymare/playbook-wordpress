# playbook-wordpress

### Objectives

Automate entire LAMPstamp project utilizing Ansible for future deployment on AWS, GCP, and Azure

### Technologies used

* **L**inux - Ubuntu Trusty Tahr 14.04
* **A**pache HTTP Server
* **M**ySQL RDBMS
* **P**HP Hypertext Preprocessor
